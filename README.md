# Test premo
<p>This test sample is created using ReactJS as UI library, also the setup was created using [Vite](https://vitejs.dev/)</p>

<h3>Project setup</h3>

<p>Install debendencies via:</p>

```
$ cd <project_root>
$ npm install --save
```


<h3>Usage</h3>

<p>Once setup is done, just run:</p>

```
$ npm run dev
```
